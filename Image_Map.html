Image Map

The HTML map tag defines an image map. An image map is an image with clickable areas. The areas are defined with one or more tags.

BackGround Image

A background image for p element

You can specify background images for any visible HTML element.
In this example, the background image is specified for a p element.
By default, the background-image will repeat itself in the direction(s) where it is smaller than the element
where it is specified. (Try resizing the browser window to see how the background image behaves.
